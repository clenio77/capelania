## ÉPICO 1: Estrutura e Navegação Base
**Objetivo:** Criar a estrutura fundamental do portal com navegação intuitiva

### US-001: Página Inicial Atrativa
**Como** visitante do portal  
**Eu quero** ver uma página inicial com carrossel das comunidades e informações principais  
**Para que** eu tenha uma primeira impressão impactante e encontre rapidamente o que procuro

**Critérios de Aceitação:**
- [ ] Carrossel funcional com 4 slides (uma para cada comunidade)
- [ ] Mensagem do Pároco em destaque
- [ ] Próximos eventos visíveis
- [ ] Últimas notícias em formato de cards
- [ ] Links rápidos para seções principais
- [ ] Design responsivo (mobile, tablet, desktop)
- [ ] Tempo de carregamento < 3 segundos

**Prioridade:** Alta  
**Estimativa:** 8 pontos

---

### US-002: Menu de Navegação Principal
**Como** visitante do portal  
**Eu quero** navegar facilmente entre as seções principais  
**Para que** eu encontre rapidamente qualquer informação

**Critérios de Aceitação:**
- [ ] Menu responsivo (hamburguer no mobile)
- [ ] Links para: Início, A Capelania, Comunidades, Celebrações, Notícias, Pastorais, Contribua, Fale Conosco
- [ ] Menu fixo no topo ao rolar página
- [ ] Indicador visual da página atual
- [ ] Funciona em todos os navegadores modernos

**Prioridade:** Alta  
**Estimativa:** 5 pontos

---

### US-003: Rodapé Informativo
**Como** visitante do portal  
**Eu quero** ver informações de contato e links importantes no rodapé  
**Para que** eu possa encontrar informações mesmo após rolar a página

**Critérios de Aceitação:**
- [ ] Informações de contato (endereço, telefone, email)
- [ ] Links para redes sociais
- [ ] Links para páginas importantes
- [ ] Copyright e informações legais
- [ ] Design consistente com o restante do site

**Prioridade:** Média  
**Estimativa:** 3 pontos
